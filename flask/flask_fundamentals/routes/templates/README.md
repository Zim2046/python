# Flask Template

- [ ] make a new directory for our application

```bash(Terminal)
mkdir flask_app
cd flask_app
```

- [ ] create the virtual environment

```bash(Terminal)
pipenv install flask
pipenv shell
```
- [ ] create [`server.py`](server.py)
```bash
touch server.py
```
- [ ] start the server

```
python3 server.py
```
